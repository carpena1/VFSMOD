Binaries for VFSMOD and UH
--------------------------
Depending on the application (inverse, SWAN, VFSMOD-GUI, etc.) the binaries for Mac and windows will need to be rename on the final destination, i.e.
(Windows): vfsmxxx.exe --> vfsm.exe ; uhxxx.exe --> uh.exe
(Mac): vfsm4xxx --> vfsm ; uhxxx --> uh; Note: 2 versions are provided for intel based Mac _x64, and new Silicon M processors _arm64)
